# Makine Öğrenmesi ile Kredi Kartı Dolandırıcılığı (Fraud) Tespiti

Bu proje, finansal işlemlerdeki dolandırıcılık (fraud) vakalarını anomali tespiti ve sınıflandırma algoritmaları kullanarak önceden yakalamayı amaçlayan uçtan uca bir veri bilimi (Data Science) projesidir.

## Kullanılan Teknolojiler & Konseptler
- **Python:** Pandas, NumPy, Matplotlib, Seaborn
- **Gelişmiş Makine Öğrenmesi (Scikit-Learn):** Random Forest Classifier, Stratified Train-Test Split
- **Veri Bilimi Metrikleri:** Imbalanced Data (Dengesiz Veri) Yönetimi, Confusion Matrix, Precision, Recall, F1-Score

## Proje Çıktıları
- Gerçek hayattaki gibi sadece %3 fraud içeren oldukça dengesiz bir veri seti simüle edilmiştir.
- Model eğitilirken veri dengesizliğini çözmek adına stratejik bölme teknikleri uygulanmıştır.
- Geliştirilen Random Forest modeli yardımıyla yüksek Precision ve Recall oranlarına ulaşılarak finansal kayıpları engelleyecek bir anomali tespit sistemi kurulmuştur.
